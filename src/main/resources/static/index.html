Welcome to Tournament Management System

Description

The objective of this application, Tournament Management System, is to create a web application for managing football tournaments, teams, matches, and scores. The application allows users to perform various actions such as creating tournaments, adding teams to tournaments, scheduling matches, updating match scores, and more

Tournament Management: Admin users can create new tournaments, edit existing tournaments, and delete tournaments. Team Management: Teams can be created, edited, and deleted. Admin users can also assign teams to tournaments. Match Scheduling: Matches can be scheduled within tournaments, with options to set the date, time, and location. Score Management: Amin users can update match scores.

Application Workflow

1. Use to Tournament menu to check existing tournament, create new one.

2. Register teams before adding them to the tournament as participants. Adding team to tournament can be done on tournament details page.

3. Schedule match from tournament details page.

4. Menu for teams and match allow user to see list of al registered teams and matches. User can access details for team or match.

5. A team should be removed from tournament before it can be deleted.
